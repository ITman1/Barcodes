================================================================================
                             OPENCV LIBRARY
================================================================================

Inside this directory are placed prebuilt OpenCV libraries for common mobile 
Android platforms (libs directory) and include header files (include directory).
All necessary for building the Barcodes Library for Android. 

Inside this directory is also located makefile (OpenCV_Android.mk) which 
installs prebuilt libraries into Android project.

See: http://opencv.willowgarage.com
